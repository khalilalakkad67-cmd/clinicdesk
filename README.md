# ClinicDesk - Clinic Management Dashboard

ClinicDesk is a PHP & MySQL clinic management dashboard for managing users, doctors, appointments, prescriptions, and reports.

## Technologies Used

- PHP
- MySQL
- HTML / CSS
- AdminLTE 3
- XAMPP
- MVC Pattern

## Main Features

- Login and logout system
- Role-based access control: Admin, Doctor, Patient
- Admin, Doctor, and Patient dashboards
- Users management with search, role filter, and pagination
- Doctors management
- Specializations management
- Appointment booking with conflict check
- Appointment filtering and pagination
- Appointment status update
- Prescription creation
- PDF prescription upload
- Secure PDF prescription download
- Patient prescription history
- Appointment reports
- CSV export
- CSRF protection
- Password hashing
- Prepared statements
